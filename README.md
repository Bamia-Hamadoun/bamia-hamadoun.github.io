## 🔄 Mettre à jour ton site en ligne

1. Va sur ton dépôt GitHub
2. Clique sur `index.html` → icône ✏️ (crayon)
3. Sélectionne tout (Ctrl+A) → Suppr
4. Ouvre le nouveau `index.html` (dans le Bloc-notes ou VS Code), Ctrl+A, Ctrl+C
5. Colle dans l'éditeur GitHub (Ctrl+V)
6. Descends → « Commit changes »
7. 1–2 minutes plus tard → rafraîchis ton site ✨

## 📄 Version PDF

`Ctrl/Cmd + P` → « Enregistrer en PDF » pour avoir une version papier propre (la section Review CV et les éléments décoratifs disparaissent automatiquement).
